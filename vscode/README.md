# WC Live - World Cup Live Text for VS Code

在 VS Code 状态栏实时显示世界杯文字直播。

## 功能

- ⚽ 状态栏显示：比分 | 比赛时间 | 解说文字
- 🔄 每 10 秒自动刷新
- 📝 点击状态栏可选择：刷新、切换比赛、打开终端

## 安装

1. 复制 `~/.wc-live.json` 配置文件（已存在于用户目录）
2. 运行 `npm install` 安装依赖
3. 按 F5 启动调试模式

## 配置

配置文件 `~/.wc-live.json`：

```json
{
  "saishi_id": "1869199"
}
```

## 使用

- 状态栏左侧会显示比赛信息
- 点击状态栏可进行操作
- 运行 `wc-live` 命令可打开终端版直播
